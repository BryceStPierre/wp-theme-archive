== Description ==

A simple but flexible WordPress theme built using Bootstrap 4.3.

== Frequently Asked Questions ==

= Does this theme include Bootstrap embedded in the theme? =
Yes, the theme contains Bootstrap, its minified CSS and JS (including Popper.js), as well as the required jQuery library.

= Does this theme include Font Awesome icons? =
Yes, the theme has Font Awesome Free 5.9.0 icons embedded.
